## E-commerce Business Intelligence

Dashboard executivo desenvolvido no Power BI para análise de 
indicadores estratégicos de e-commerce.

### Indicadores
- Receita Total, Margem de Lucro, Lucro Total
- Clientes ativos e Quantidade Vendida
- Análise por país, categoria e método de pagamento

### Tecnologias
- Power BI
- DAX
- Modelagem dimensional (Star Schema)
